-> Start Up
Before beginning
1. npm i
2. touch .env and add TOKEN={YOUR TOKEN} to .env

> Development
1. npm run test OR npm run dev

> Production
To start
1. npm run start

To stop
1. npm run stop

-> Dependencies
Node v12, https://nodejs.org/
Forever, https://www.npmjs.com/package/forever
